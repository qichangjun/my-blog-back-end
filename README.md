# 小七博客后台仓库

 博客访问地址 : [http://116.62.227.214](http://116.62.227.214).

## 本地启动 

 `git-clone http://**` <br>

 `npm install ` -- 安装依赖包<br>

 `npm start` --启动服务<br>

 访问`locathost://3100` 

## 正式环境打包

输入 `npm run prd`

## 后台环境

`node@8.4.0` <br>

`koa2` <br>

... <br>

详细见`package.json`文件

 **登陆注册** <br>
 **发表，编辑，删除文章** <br>
 **个人管理，管理员的平台管理** <br>
